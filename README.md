# Kanishka Agarwal Portfolio

Dark, high-conviction marketing portfolio for B2B SaaS founder audiences.

Live site:

- https://kanishka-ag.netlify.app/

## Overview

This repository contains a static portfolio site built to position Kanishka Agarwal as a Head of Marketing for early-stage and growth-stage B2B SaaS companies.

The site focuses on:

- strategic clarity
- case-study-led proof
- premium dark visual design
- strong conversion CTAs

## Stack

- HTML
- CSS
- Vanilla JavaScript

## Main Files

- `index.html` — current portfolio site
- `orion-ga4.png` — Orion GA4 case study visual
- `orion-ga4.jpeg` — alternate image asset

## Local Preview

Open `index.html` in a browser, or serve the folder with any static server.

## Deployment

The site is currently deployed on Netlify.
